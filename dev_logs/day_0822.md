### cd 내용
- main, develop 브랜치별 cd스크립트 분리.
    - main 브랜치 ( push | pull-request )
        - build
            - create zip
            - upload s3
        - deploy
            - update lambda from s3
    - develop 브랜치 ( push | pull-request )
        - package requirements
        - required files exist
        - aws cli credential
